# TechStore

Учебный проект интернет-магазина электроники.

## Что сделано

- Базовая структура проекта
- Главная страница (HTML/CSS/JS)
- Подключен Git и GitHub
- Настроены Prettier и Live Server

## Технологии

- HTML
- CSS
- JavaScript
- Git
- VS Code

## Как запустить

1. Скачать проект
2. Открыть папку в VS Code
3. Установить Live Server
4. ПКМ по index.html → Open with Live Server
